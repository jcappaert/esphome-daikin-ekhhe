# Examples

The example configurations in this directory are intended as starting points:

- `minimal.yaml`: small everyday setup with key readings and controls.
- `full.yaml`: broader normal-user setup exposing most supported entities.

Use the examples as references, then copy only the entities you want into your own ESPHome node. The examples assume an ESP32-C3 with UART on `GPIO20`/`GPIO21`; adjust pins, board, Wi-Fi, and node names for your own hardware.
